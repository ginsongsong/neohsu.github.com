How to Install Sun JDK 6 on Ubuntu

Outline

  • JDK 提供開發人員的軟體開發工具並建置編譯與執行 JAVA 的開發環境
  • JDK 包含了 JAVA 開發工具外，還包含了 JRE 和 JVM
  • JDK 可以開發與運行 JAVA 程式

Development environment

  • Ubuntu 11.10 (x86-64)
  • Dell Studio XPS 1640
